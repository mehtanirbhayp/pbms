Create a complete backup of all your data including companies, customers, loans, receipts, and users. This backup file can be used to restore your data on another computer or as a safety backup.

Warning: Importing data will replace ALL existing data in the system. This action cannot be undone. Make sure you have a backup before proceeding.

📋 Instructions

  • Import: Select a previously exported backup file and click "Import Data" to restore it.
  • Migration: To migrate to another computer, export on the old system and import on the new system.
  • Backup: Regularly export your data as a safety backup.
